Liten

A free, fast and accurate duplicate file finder for Macs.

Download this project as a .zip file Download this project as a tar.gz file

Liten is open source deduplication software, with an advanced identification and search engine. Use it to drive your utility apps for commercial or non-commercial purposes.

And keep us posted on the progress! Thanks.

Authors and Contributors

In 2010, Noah Gift (@noahgift) developed Liten, which was launched shortly after as Zebulant's (www.zebulant.com) first product with fellow UC Davis MBA's David Thayer (@DavidThayer) and Cliff McEnery.
